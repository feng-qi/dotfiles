Installation:

	git clone https://github.com/feng-qi/dotfiles.git ~


Create symlinks:

```sh
ln -s ~/dotfiles/vimrc ~/.vimrc
ln -s ~/dotfiles/gvimrc ~/.gvimrc
ln -s ~/dotfiles/UltiSnips/ ~/.vim/UltiSnips

ln -s ~/dotfiles/agignore ~/.agignore
ln -s ~/dotfiles/zshrc ~/.zshrc
```

file      | program
--------- | --------
agignore  | ag
